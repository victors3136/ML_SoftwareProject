# SpotifySongsClustering for Machine Learning
## Setup instructions
After cloning the repo, make sure you have at least python 3.14 installed.
Having solved that, run the following commands in the terminal:  
 - `python -m venv .venv`
 - `.\.venv\Scripts\activate`
 - `python -m pip install -e .`
 - `pip install -qr requirements.txt`
